# AI&U -- Thumbnail and Branding Guidelines

## Brand Intent

AI&U should feel like practical GenAI presented through a clean learning interface. The visual identity communicates calm authority, clarity, modernity, structured thinking, and approachability. It should not feel loud, flashy, overdesigned, or chaotic. The guiding visual idea is high-clarity educational UI with deliberate accents.

## Core Visual System

### Base Colours
- Cloud White background
- Ink Black primary text
- Slate secondary text
- Mist outlines and dividers

### Accent Colours by Pillar
- Pillar 1 (AI for Common Person): Amber Orange
- Pillar 2 (Using AI Tools): Signal Green
- Pillar 3 (Building AI Workflows): Signal Gold

### Status Logic
- Green: success, action, "do this"
- Gold: key idea, rule, highlight
- Cocoa: warning, trap, "do not do this"

Each asset should typically use neutrals plus one primary pillar accent, with an optional status colour if needed. The system must remain restrained.

## Typography

- **Headlines and thumbnail text:** Space Grotesk (strong, modern, readable at small sizes)
- **Body, labels, interface text:** Inter (clean, professional)
- **Layout personality:** Card-based composition, generous white space, rounded rectangles, chips and tags, progress bars, clear visual grouping

## Thumbnail Templates

### Template A: Big Promise + Proof
Use when the video offers a strong direct payoff. Structure: face or character on one side, big 2-4 word claim on the other, one badge or small supporting UI card.

### Template B: Before vs After
Use when contrasting manual vs AI, chaos vs system, or bad workflow vs better workflow. Structure: split layout, arrow or movement cue, one highlighted transformation.

### Template C: Workflow Diagram
Use for system or architecture videos. Structure: 3-5 nodes, arrows, one node highlighted, short title in corner.

## Thumbnail Rules

- Few words (2-4 maximum)
- Strong readability on mobile (test at small size)
- No clutter -- one primary focal point
- Packaging must match the actual video promise (no clickbait drift)
- One emotion + one object + 2-4 words is the formula

## Video Graphics Package

The channel maintains a reusable graphics kit for consistency and speed:
- Intro sting (branded opening)
- Lower thirds (name/title overlays)
- Chapter cards (section transitions)
- Callout arrows and focus rings (attention direction)
- End-screen frame (outro with next video routing)
- Checklist cards (summary takeaways)
- Workflow node graphics (system diagrams)

## Visual Asset Philosophy

Visuals are a teaching layer, not decoration. Every visual should simplify a complex idea, create a pattern interrupt, show a workflow or relationship, reduce explanation time, or increase retention through clarity. Visual categories include static infographics, motion loops, interactive explainers recorded into the edit, thumbnail assets, and UI-mockup-style teaching visuals.
