## Template: Relationship Entry

Each entry follows this format:

- **Relationship:** [colleague / mentor / collaborator / client / advisor]
- **Organization:** [company or affiliation]
- **Context:** [how the relationship was formed, shared projects, key interactions]
- **Communication cadence:** [weekly / monthly / quarterly / as-needed]
- **Key notes:** [preferences, important dates, topics of shared interest]
