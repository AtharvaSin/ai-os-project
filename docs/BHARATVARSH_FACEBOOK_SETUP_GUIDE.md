## How Facebook Differs from Instagram

| Dimension | Instagram | Facebook |
|-----------|-----------|----------|
| Primary purpose | Visual discovery + new audience | Community + discussion + retention |
| Algorithm priority | Sends (DM shares), watch time | Comments, shares, meaningful interactions |
| Content format | Reels-first, carousels | Discussion posts, video, Events, Live |
| Growth engine | Explore page, Reels tab | Groups, shares, Events, cross-posting |
| Audience behaviour | Browse, save, share via DM | Comment, debate, join groups, attend events |
| Link handling | Bio only (no clickable links in posts) | Clickable links in posts (but deprioritized) |
| Asset reuse from IG | — | ~60% of Instagram assets can cross-post with adapted captions |

**Key insight:** Instagram gets people into the world. Facebook keeps them talking about it.
