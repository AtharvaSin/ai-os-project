## How Twitter/X Differs from Instagram & Facebook

| Dimension | Instagram | Facebook | Twitter/X |
|-----------|-----------|----------|-----------|
| Primary purpose | Visual discovery | Community + discussion | Reach + real-time conversation |
| Growth format | Reels | Groups + Events | Threads + Replies |
| Algorithm priority | Sends, watch time | Comments, shares | Replies, bookmarks, early engagement |
| Content style | Polished visual | Conversational long-form | Sharp, opinionated, fast |
| Hashtags | 5-8 per post | Minimal | 1-2 max (more gets penalized) |
| Character limit | 2200 (caption) | 63,206 | 280 (standard) / 25,000 (Premium) |
| Best for Bharatvarsh | World aesthetics | Reader community | Lore drops, provocative takes, virality |
| Asset reuse | Source platform | ~60% from IG | ~40% visuals from IG + unique text |

**Key insight:** Instagram shows the world. Facebook discusses it. Twitter/X makes it go viral.

- **Dark mode:** On (matches the Bharatvarsh aesthetic in screenshots)
